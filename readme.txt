Inglês:
This plugin adds installments quantity and the price of each installment, on WooCommerce loop pages and single product page.

Settings:
After plugin activation go on "Settings" > "Installments" and check the box to enable plugin functionality.
On the same page you can define installments quantity, minimum value for each installment, a prefix text and a suffix text.


Portugês:
Este plugin para WooCommerce adiciona quantidade de parcelas e o valor de cada parcela, nas páginas que listam todos os produtos e na página individual de cada produto.

Configuração:
Depois de ter ativado o plugin navegue em "Configurações" > "Parcelas" e marque a caixa de seleção para habilitar a funcionalidade do plugin.
Nessa mesma página é possível definir a quantidade de parcelas, um valor mínimo para cada parcela, um texto prefixo, que virá logo antes da quantidade, e um texto sufixo, que virá logo depois do valor do preço.
